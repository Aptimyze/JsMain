To enable JavaScript, follow the instructions below for your browser

Internet Explorer 5 or higher:

  1. Click Tools, then click Internet Options.
  2. Click the Security tab.
  3. Click the Custom Level button.
  4. Scroll to the Scripting section.
  5. Check Enable under Active scripting, Allow paste operations via script, and Scripting of Java applets.
  6. Click the OK button.

Internet Explorer 4.x:

  1. Click View
  2. Click Internet Options.
  3. Click the Security tab.
  4. Click the Settings button.
  5. Scroll to the Scripting section.
  6. Check Scripting of Java applets and Active Scripting.
  7. Click the OK button.

Netscape 6.x:

  1. Click Edit, then click Preferences
  2. Click on Advanced.
  3. Check Enable JavaScript for Navigator.
  4. Click the OK button.

Netscape 4.x:

  1. Click Edit, then click Preferences
  2. Click on Advanced.
  3. Check Enable JavaScript.
  4. Click the OK button.

Mozilla FireFox:

  1. Select Tools
  2. Click on Options
  3. Click Web Features.
  4. Click Enable Java.
  5. Click Enable JavaScript.
