Class ContactHandler

Description
Description | Methods (details) Constants (details)

Contact Handler Class which holds all the information required by libraries to perform any action related to contact engine.

* How to call this class.

  1.  $viewerObj=new Profile(144111);
  2.  $viewedObj=new Profile(136580);
  3.  $engineType=ContactHandler::EOI;
  4.  $contactObj=new  Contacts(144111,136580);
  5.  $toBeType="I";
  6.  $action=ContactHandler::PRE;
  7.  $contactHandler=new ContactHandler($viewerObj, $viewedObj, $engineType, $contactObj, $toBeType, $action);
