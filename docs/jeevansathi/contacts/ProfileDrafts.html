CLASS ProfileDrafts ProfileDrafts class fetch all the drafts that user has saved while sending EOI or while accepting/declining user.

* Code to fetch user's drafts

  1.  $draftsObj=new ProfileDrafts(Profile $profileObj);
  2.  //to fetch accepted drafts
  3.  $draftsObj->getAcceptDrafts();
  4.  // to fetch decline drafts
  5.  $draftsObj->getDeclineDrafts();
  6.  // to fetch Eoi drafts
  7.  $draftsObj->getEoiDrafts();
